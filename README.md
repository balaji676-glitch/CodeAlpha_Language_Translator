## Project Overview

This project is an AI-powered Language Translation Tool developed as part of the CodeAlpha Artificial Intelligence Internship. The application allows users to input text, select source and target languages, and receive translated output instantly using an AI translation API.

## Features

* Text translation between multiple languages
* Simple and responsive web interface
* Real-time translation using API integration
* Easy-to-use design suitable for beginners

## Technologies Used

* Python
* Flask (Web Framework)
* HTML & CSS
* Translation API (LibreTranslate / Argos Translate)

## How to Run the Project

1. Install required libraries:

```
pip install flask requests
```

2. Run the application:

```
python app.py
```

3. Open browser and visit:

```
http://127.0.0.1:5000
```
